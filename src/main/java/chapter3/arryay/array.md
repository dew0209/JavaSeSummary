数组的基本概念和使用

```xml
数组是一种数据结构，用来存储同一类型值的集合。通过下标可以访问数组中的每一个值。
在声明数组变量时，需要指出数组类型(数据元素类型紧跟[])和数组变量的名字。
可以有以下两种方式声明一个数组：
	int a[];
	int[] a;
不过，这条语句只声明了变量a，并没有进行初始化操作。
int[] a = new int[100];
常见概念：
	数组名：a
	下标[索引]：0~99，设数组的长度为n，则下标的范围为[0,n - 1]
	元素：同一类型
	数组的长度：100
注意：
	1.创建一个数字数组时，所有元素都初始化为0。boolean数组的元素会初始化为false。对象数组则初始化为一个特殊值null，这表示这些元素还未存放任何对象。
	2.一旦创建了数组，就不能再改变它的大小。
	3.创建数组对象会在内存中开辟一块连续的空间，而数组名中引用的是这块连续空间的首地址。
```

for each循环

```xml
Java有一种功能很强的循环结构，可以用来依次处理数组中的每个元素(其他类型的元素集合亦可)，而不必为指定下标值而分心。
格式：
	for(variable : collection)statement
注：使用传统的for循环亦可遍历
```

数组初始化及匿名数组

```xml
在Java中，提供了一种创建数组对象并同时赋予初始值的简化书写操作。
	int[] smallPrimes = {2,3,5,7,11,13};
甚至还可以初始化一个匿名数组
	new int[]{17,19,23,29,31,37};
使用这种方法可以在不创建新变量的情况下重新初始化一个数组。
注：在Java中，允许数组长度为0.在编写一个结果为数组的方法时，如果碰巧结果为空，则这种语法形式就有用了。此时可以创建一个长度为0的数组：
	new elementType[0];数组长度为0与null不同
```

数组拷贝

```xml
在Java中，允许将一个数组变量拷贝给另外一个数组变量。这时，两个变量将引用同一个数组。
如果希望将一个数组的所有值拷贝到一个新的数组中去，就要使用Arrays类的copyOf方法：
	int[] newArr = Arrays.copyOf(arr,arr.length);//第二个参数是新数组的长度
这个方法通常用来增加数组的大小。
	int[] arr = Arrays.copyOf(arr,2 * arr.length);//多余的元素的值根据类型决定
```

多维数组

```xml
多维数组将使用多个下标访问数组元素，它适用于表示表格或更加复杂的排列形式。
声明：
	int[][] balances;
同一维数组，在初始化之前不能使用它。
初始化：
	balances = new int[3][4];//创建一个三行四列的数组
	亦或者
	int ac = {
		{1,2,3},
		{1,2,3}
	};//知道数组元素
访问：balances[i][j]
```

不规则数组

```xml
实际上，没有多维数组，只有一维数组。多维数组被解释为：数组的数组
创建一个不规则数组：数组的每一行有着不同的长度
格式：
	int[][] odds = new int[11][];
  for(int n = 0;n < 11;n++){
      odds[n] = new int[n + 1];
  }
```
